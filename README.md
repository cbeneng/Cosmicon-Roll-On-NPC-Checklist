# Cosmicon-Roll-On-NPC-Checklist
Very simple checklist program for users to keep track of NPC's battled in Honkai Star Rail's "Cosmicon, Roll on!" Event.

NPC Numbers and Areas are correlated to this guide/map by game8: https://game8.co/games/Honkai-Star-Rail/archives/580541

Entire python script is in main.py so you will have to read and compile yourself.
